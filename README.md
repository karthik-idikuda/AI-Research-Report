# AI Research Report Generator

Automated research paper summarizer and report generator that extracts key findings, methodology, and conclusions from academic papers.

## Features

- PDF and text-based paper ingestion
- Automatic extraction of abstract, methodology, and conclusions
- Multi-paper comparative analysis
- Structured report generation in Markdown and PDF
- Citation management and bibliography generation

## Tech Stack

TypeScript, NLP, Transformers, Node.js, PDF.js

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Research-Report.git
cd AI-Research-Report
npm install
```

### Usage

```bash
npm start -- --input paper.pdf
```
